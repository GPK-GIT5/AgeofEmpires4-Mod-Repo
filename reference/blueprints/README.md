# AoE4 Blueprint Reference

Blueprint data extracted from the Attributes dump for use in SCAR scripting.

## Blueprint Types

| Type | Code | Description |
|------|------|-------------|
| **SBP** | Squad Blueprint | Units (soldiers, villagers, traders, ships) |
| **EBP** | Entity Blueprint | Buildings, walls, landmarks, siege (placed entities) |
| **UPG** | Upgrade Blueprint | Technologies, age-ups, unique upgrades |
